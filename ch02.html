I/O redirection

  • $ Process has input, output, error output
  • – Process : running program
  • $ I/O redirection changes input/output
  • $ Standard input : keyboard (0)
  • $ Standard output : screen (1)
  • $ Standard error output : screen (2)
  • > or 1> : change output
  • 2> : change error output
  • < : change input

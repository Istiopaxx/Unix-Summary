Environment Variable

  • $ Affects the look and feel of computing experiences
  • $ PS1 : commands prompts
  • – $PS1=‘[hello >> ‘ // sh
  • – $PS1=“[\h\w \# >> “ // bash
  • $ PATH : contains a list of directory that are searched when a user input a command
  • – PATH=/usr/bin:/usr/ucb
  • – PATH=$PATH:/usr/sbin

Local/environment variable

  • $ Local variable
  • – Affects to current shell only
  • – $ a=3
  • $ environment variable
  • – Affects to sub-shell, too
  • – $ export a
  • $ Example
  • – $ a=3; b=5
  • – $ echo $a $b // good
  • – $ export b
  • – $ sh // start sub-shell
  • – $ echo $a // no result
  • – $ echo $b // 5
  • – $ exit // return to parent(original) shell

Variable commands

  • $ set
  • – See all variables and its value
  • – $ set | grep PATH
  • $ env
  • – See all environment variables and its value
  • $ echo $variable
  • – $ echo $a
  • $ export
  • – makes a viable to an environment variable
  • – $ export a
  • $ Unset
  • – $ unset a

System/user variable

  • $ System variable
  • – Embedded variable
  • – PS1, PATH, SHELL, Lang, …
  • $ User variable
  • – User defined variable
  • – $ a=3

Absolute/Relative Path

  • $ Absolute Path
  • – Starts with / (root) path
  • – /usr/bin/ls
  • $ Relative Path
  • – Doesn’t start with /
  • – Reference is current directory
  • – dir1/hi.txt
  • $ Shorthand notations
  • . : current directory (./.profile)
  • .. : parent directory (../dir2/hi.txt)
  • ~ user’s home directory (~unix300/hi.txt)

Kinds of Shell

  • $ Bourne Shell (sh)
  • – Original unix shell
  • $ C Shell (csh)
  • – Syntax is similar with C programming language
  • $ Korn Shell (ksh)
  • – sh + csh + additional functions
  • $ Bourne Again Shell (bash)
  • – sh + many functions

Shell initialization files

  • $ /etc/profile // global configuration file
  • $ Sh
  • – ~/.profile
  • $ Csh
  • – ~/.cshrc, ~/.login
  • $ Ksh
  • – ~/.profile, ~/.kshrc
  • $ Bash
  • – ~/.bashrc_profile

alias

  • $ Substituting one term to another
  • $ alias
  • $ alias ls=‘ls –l’
  • $ unalias ls
  • $ Sh
  • – $ dir() ls –l $*
  • – $ dir /

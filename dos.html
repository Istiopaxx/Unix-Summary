DOS

  • Disk operating system
  • Interpreter between user and computer
  • Starting DOS
  • – start->all programs->aux programs -> command prompt
  • – Start->at cmd window ->cmd->Enter
  • – At File Explorer -> (Shift + right-click at specific directory)

Dos commands

  • Internal commands
  • – Memory-resident
  • – ver, vol, date, time, cls, dir, md, cd, path, rd, copy con, type, copy, del, ren, prompt
  • External commands
  • – Disk-resident
  • – Xcopy, move, fc, doskey, mem, more, sort, find, attrib, deltree, edit, tree
  • Capital or Small letters will do

Dos commands

  • Ver : version number
  • Vol : volume label
  • Date : display/set current system date
  • Time : display/set current system time
  • cls : clear screen
  • Dir : list of directories and fil
  • – Switch : /R /P /W /B /L /A
  • Path : defines a list of directories DOS searches for external commands
  • Rd(rmdir) : delete the empty directory
  • Md(mkdir) : make a new directo
  • Type : see the contents of an existing file
  • Copy : copy a file
  • Del/erase : delete a file
  • Ren : change the file name
  • Tree : display directory structure
  • Move : moves file(s)
  • Fc : file compare
  • Doskey : remember command history – Use up/down key
  • Mem : display total available memory
  • Attrib : see attributes of a file
  • Edit : start dos editor
  • More : see page by page – More aa.txt – Dir | more
  • Sort : arrange data in an order – Sort aa.log – Sort aa.log > b.log
  • Find : display lines that has the string – Find “201503” *.log – Dir | fine “txt”

Dos commands

  • Prompt : set prompt
  • – Prompt=$$ -> $
  • – Prompt=$p$g -> pwd>
  • Copy con : create a new file
  • – Copy con aa.txt
  • – Hi // contents
  • – Hello // contents
  • – ^z // end of file

Unix Command Syntax

  • $ command [option] [arguments]
  • $ ls –sCF /usr/bin
  • $ : prompt
  • -sCF : options || same with –s –C –F
  • /usr/bin : argument
  • [ .. ] : optional
  • Usage : $ man ls

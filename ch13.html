Unix Summary - Chapter 13

Script Language

  • Script is a program runs on the interpreter
  • Unix Shell is an interpreter
  • – DOS Shell is an interpreter, too.
  • Shell reads the script and runs it directly line-by-line
  • <-> Compile language like C/C++, must be compiled before running

Usage

  • Shell Program (scripts) is used for System administration
  • Automate a vast of range of jobs with a few simple scripts
  • – Boring and repetitive jobs

Script file

  • #!/bin/sh
  • – Executed by /bin/sh
  • – Shebang : hash(#), bang(!)
  • # : comments
  • Variable=value
  • – a=1
  • – a=`expr $a+1` // back quote
  • read var
  • – Read from keyboard and save to variable var

Special Variables

  • ? : previous command’s exit status
  • $ : PID if the current shell
  • - : Options at the start-up
  • ! : PID of the last background process
  • 0 : script name
  • 1-9 : command-line arguments 1-9
  • - : last argument

Flow Control

  • Test command
  • – test expression
  • – [ expression ]
  • Operator : =, <, >, !=
  • Conditional Flow Control
  • – If-then-fi, if-then-else-fi, case-in-esac
  • Iterative Flow Control
  • – While-do-done, until-do-done

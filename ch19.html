Open Source Licensing

  • BSD License
  • – Source code can be modified
  • – Modification do not have to be redistributed
  • – Advertised as BSD code is used
  • – Used as proprietary or commercial software
  • GNU Public License (GPL)
  • – Called as "copylight license"
  • – Any modified version of the distributed software can be released under the same GPL terms (-> termed as not "open source" but "free software")

Install Lynx software

  • Download from lynx.isc.org
  • Unzip
  • PATH=$PATH:/usr/sfw/bin:/usr/ccs/bin:.
  • Export PATH
  • ./configure
  • Make
  • Make install
  • Run Lynx

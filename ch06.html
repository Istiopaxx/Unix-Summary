Unix Summary - Chapter 6

Anatomy of a Command

  • $ Command [options] [arguments]
  • – $ ls –sCF /usr/bin
  • – $ ls –s –C –F /etc
  • $ man ls
  • $ apropos ls
  • – Find related files
  • $ whereis echo
  • $ which echo

Meta Characters

  • $ Wildcard
  • – ? : matches any one character
  • – *: matches character or characters
  • – [] : matches one of the characters in […]
  • $ ls *.txt
  • $ ls hi?.txt
  • $ ls hi[12357].txt

Redirection

  • $ ls > ls.txt
  • $ sort < /etc/group
  • $ sort < /etc/group > sort.txt
  • $ mkdir dir1
  • $$ mkdir dir1 2> err.txt
  • $ find / -name "hi.txt" 2> /dev/null
  • $ find /export/home –name "hi.txt" > out.txt 2>&1
  • $ find /export/home –name "hi.txt" 2> out.txt 1>&2

Pipes

  • $ Who | wc
  • $ Cat /etc/passwd | grep unix30
  • $ Echo $PATH
  • $ Echo ${PATH}${TERM}
  • $ Echo “$PATH”
  • $ Echo "$PATH"
  • $ Echo "\HPATH:
  • $ Echo today is `date`

Quotas

  • $ A limit on the amount of disk space each user is allocated
  • $ /etc/vfstab enable quotas option
  • – (usrquota,grpquota) should be set
  • $ quotacheck –acug /export/home
  • $ cd /export/home
  • $ touch quotas
  • $ chmod 600 quotas
  • $ edquota user-id
  • – soft, hard, i-nodes

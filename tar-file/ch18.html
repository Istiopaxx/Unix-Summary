Unix Summary - Chapter 18

Backup Media Types

  • Magnetic Tapes
  • CD/DVD-ROM
  • Zip drives
  • Hard drive/network backup
  • USB

Backup Types

  • Full backup
  • – entire
  • Differential Backup
  • – that has changed since last full backup
  • Incremental Backup
  • – that has changed since the completion of any other type of backup

tar

  • Tape archive
  • c : creates
  • t : shows contents
  • x : extracts
  • v : verbose
  • f : set filename
  • tar cvf mydir.tar mydir
  • tar tvf mydir.tar
  • tar xvf mydir.tar

compress

  • compress mydir.tar
  • – > Mydir.tar.Z
  • uncopmress mydir.tar
  • gzip mydir.tar
  • – > mydir.tar.gz
  • gzip –d mydir.tar.gz
  • gunzip mydir.tar.gz

example

  • $ tar cvf mydir.tar mydir
  • $ compress mydir.tar
  • $ ftp 10.10.54.75
  • Login using id/passwd
  • ftp> bi
  • ftp> put mydir.tar.Z
  • ftp > quit
  • $ telnet 10.10.54.75
  • Login using id/passwd
  • $ uncompress mydir.tar.Z
  • $ tar xvf mydir.tar
